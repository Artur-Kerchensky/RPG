Название проекта: «The Forgotten Lands»

Задание проекта: Создание компьютерной игры в жанре Top-Down RPG на основе библиотеки «Pygame».

Применяемые технологии:
Базы данных (хранение и обработка информации)
Библиотеки Python (основа – pygame, прикладные – math, random, os, sys)
Текстовые документы (обработка файлов конфигурации)
Аудиофайлы (с использованием pygame.mixer)
Математические алгоритмы (шум Перлина, «поиск в ширину»)

Архитектура проекта:
data - папка с данными игры
configuration.py - модуль с методами по управлению файлами конфигурации 
launcher.py - модуль для обработки стартового меню (лаунчера игры)
map.py - модуль с классами чанков, камеры и карты
main.py - основной модуль программы 
map_generation.py - модуль, реализующий генерацию игровой карты
object.py - модуль, содержащий классы объектов и методы их взаимодействия
database.py - модуль, предназначенный для работы с базой данных
requirements.txt - текстовый файл со списком необходимых для работы модулей
